# output/

Runtime output lands here.

A successful `npm start` writes `narration.mp3`, a short MP3 spoken in the cloned voice. The cloned voice itself is deleted from your account immediately after synthesis, so nothing persists server-side.

If the script exits with `Voice cloning isn't included in your current Speechify plan.`, your plan does not include voice cloning and no `narration.mp3` is produced. See [Speechify pricing](https://speechify.ai/pricing) for plans that include cloning.
